# 巴菲特/芒格选股准则

## 一、经济护城河 (Economic Moat)

### 护城河类型
1. **品牌护城河** - 强大的品牌认知度（如 Coca-Cola）
2. **成本优势** - 规模经济或独特技术
3. **网络效应** - 用户越多价值越高（如 Apple 生态）
4. **转换成本** - 客户难以更换供应商
5. **专利/牌照** - 法律保护的竞争优势

### 评估指标
- 市场占有率趋势
- 毛利率稳定性 (>40%)
- 定价能力
- 客户忠诚度

---

## 二、财务指标

### 必须指标
| 指标 | 要求 | 说明 |
|------|------|------|
| ROE | >15% | 股东权益回报率 |
| ROIC | >12% | 投入资本回报率 |
| 自由现金流 | 正数且稳定 | 经营现金流 - 资本支出 |
| 毛利率 | >40% | 产品竞争力 |
| 负债率 | <50% | 财务健康度 |
| 利息覆盖率 | >5x | 偿债能力 |

### 增长指标
- 营收增长率
- 每股收益增长
- 股息增长

---

## 三、估值模型

### 安全边际计算
```
内在价值 = 未来现金流折现
安全边际 = (内在价值 - 市价) / 内在价值
要求：安全边际 > 30%
```

### 简化估值
- P/E < 15（优质公司可放宽至 20）
- PEG < 1.5
- P/B < 3（ROE > 15% 时可放宽）

---

## 四、管理层评估

### 关键问题
1. 管理层是否诚实透明？
2. 是否有良好的资本配置记录？
3. 是否与股东利益一致？
4. 是否有清晰的业务战略？

### 危险信号
- 频繁并购
- 过高的高管薪酬
- 复杂的财务报表
- 频繁更换 CEO

---

## 五、能力圈

### 识别标准
- 能否用简单语言解释商业模式？
- 是否了解行业竞争格局？
- 能否预测未来5-10年？

### 避免
- 复杂金融衍生品
- 不熟悉的科技股
- 快速变化的行业

---

## 六、买入决策清单

### 必须满足
- [ ] 业务简单易懂
- [ ] 有可持续的护城河
- [ ] 管理层诚实能干
- [ ] 财务健康（ROE>15%，自由现金流为正）
- [ ] 价格有安全边际（>30%）
- [ ] 长期前景良好

### 卖出时机
- 发现判断错误
- 业务基本面恶化
- 发现更好的机会
- 价格严重高估
