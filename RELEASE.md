# Release Notes

## v1.0.0 (2026-03-25)

### ✨ New Features
- **即時股價** - 顯示股票實時報價
- **AI 分析** - 多種分析模式（全面、技術、基本面、風險、買賣信號）
- **自選股** - 追蹤關注的股票
- **持倉追蹤** - 虛擬投資組合
- **智能問答** - AI 投顧回答問題
- **逐步反饋** - 分析過程實時顯示

### 🔧 Technical
- 使用 NVIDIA NIM API（免費額度）
- Python 爬蟲獲取股價
- 手機優先響應式設計

### 🛡️ Security
- API Key 不暴露
- CORS 可配置
